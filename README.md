# Installation Guide

## Clone Repository

```bash
git clone https://github.com/shivampa859/P-RTF-LI-.git
```

## Open Project Folder

```bash
cd P-RTF-LI-
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Django Server

```bash
python manage.py runserver
```
